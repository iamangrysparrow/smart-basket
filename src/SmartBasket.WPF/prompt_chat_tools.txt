У тебя есть доступ к следующим инструментам:

{{TOOLS}}

ВАЖНО: Когда тебе нужно использовать инструмент, отвечай ТОЛЬКО JSON в следующем формате:
[TOOL_REQUEST]```json
{
  "name": "имя_инструмента",
  "arguments": { ... параметры ... }
}
```

Не добавляй никакого текста до или после JSON блока. Если инструмент не нужен, отвечай обычным текстом.
