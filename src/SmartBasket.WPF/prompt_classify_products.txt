Принципы классификации продуктов
Основной критерий — назначение и тип продукта, определяемый его природой, способом производства и условиями хранения.

Правила классификации:
1. По происхождению — продовольственные товары отделяются от непродовольственных.
2. По категориям продуктов — каждая категория объединяет товары с одинаковым функциональным назначением и способом потребления (например, все молочные продукты или все напитки).
3. По подкатегориям — внутри категорий выделяются подгруппы на основе состава, формы выпуска или специфических характеристик:
   - Для мяса — разделение по видам (птица, говядина, свинина, индейка)
   - Для молочных — по типам (молоко, кисломолочные, сыр, масло)
   - Для овощей — по способу обработки (свежие, замороженные, консервированные)
   - Для соков и напитков — по типам (вода, соки, газированные напитки)
4. Специальные категории — выделяются товары со специфическим назначением:
   - Детское питание — отдельно от общих продуктов
   - Замороженные полуфабрикаты — отдельно от готовых блюд и свежих продуктов
   - Гигиенические товары — отдельно от пищевых продуктов

Основное правило — один товар принадлежит только одной категории, чтобы избежать путаницы и облегчить поиск.

Продукт и категория - это одна и та же сущность. Продукт отличается только тем, что у него нет потомков. Продукт на самом последнем уровне иерархии.

КРИТИЧЕСКИ ВАЖНО:
- Каждый продукт из входного списка ОБЯЗАТЕЛЬНО должен быть в ответе с "product": true
- Используй существующие категории из иерархии выше, если они подходят
- Создавай новые категории только если существующие не подходят

ФОРМАТ ОТВЕТА (строго JSON):
{
  "products": [
    {"name": "Название", "parent": null или "name-родителя", "product": true/false}
  ]
}

parent - null для корневого элемента
parent - "name-родителя" для продукта или дочерней категории
product - true если это продукт (конечный элемент, лист дерева)
product - false если это категория (может иметь потомков)

В ответе сначала перечисли категории и их иерархию. Потом перечисли продукты с назначенной категорией.
