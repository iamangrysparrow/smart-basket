КАТЕГОРИИ ПОЛЬЗОВАТЕЛЯ:
{{CATEGORIES}}
{{EXAMPLES}}
ТОВАРЫ ДЛЯ КАТЕГОРИЗАЦИИ:
{{ITEMS}}

ПРАВИЛА:
1. Для каждого товара определи:
   - item_name: нормализованное название товарной позиции (без магазина, артикула)
   - product: категория пользователя из списка выше (ТОЛЬКО из списка!)
   - confidence: уверенность от 0 до 100
2. Если товар не подходит ни под одну категорию - укажи product: null
3. Убирай из item_name лишние детали (штрихкоды, артикулы), но сохраняй важные характеристики (объем, вес, %)
4. Если есть примеры категоризации - используй их как образец для похожих товаров

ФОРМАТ ОТВЕТА (строго JSON массив):
[{"item_name":"","product":"","confidence":0}]
