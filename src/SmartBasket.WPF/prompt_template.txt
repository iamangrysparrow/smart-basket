Извлеки данные чека в JSON
Правила:
- shop: название магазина (АШАН, Пятёрочка и т.п.)
- date: дата в формате YYYY-MM-DD (год {{YEAR}} если не указан)
- order_number: номер заказа (например H03255764114)
- total: итоговая сумма (число)
- items: массив товаров

Для каждого товара:
- name: название
- quantity: количество (число)
- unit: единица измерения за цену (штуки (шт) / килограммы (кг) / литры л)
- price: цена за единицу
- amount: цена за позицию

price - первая цена
amount - вторая цена
quantity - после названия перед price
unit - после названия и quantity перед ценой по умолчанию ставь шт если не смог определить.

ИГНОРИРУЙ другие строки кроме товарных цен.
ИГНОРИРУЙ строки: "Собрано", "Оплата", "Доставка", "Сервисный сбор", телефоны, адреса.

Считай total по amount
order_datetime - дата со временем. YYYY-MM-DD:hh:mm. Если не знаешь год - {{YEAR}} по умолчанию.

JSON:
{"shop":"","order_datetime":"","order_number":"","total":0,"items":[{"name":"","quantity":1,"unit":"шт","price":0,"amount":0}]}
