Выбери лучший товар из результатов поиска в интернет-магазине.

## Что ищет пользователь
Название: {{DRAFT_ITEM_NAME}}
Требуемое количество: {{DRAFT_ITEM_QUANTITY}} {{DRAFT_ITEM_UNIT}}
Категория: {{DRAFT_ITEM_CATEGORY}}

## Результаты поиска в магазине {{STORE_NAME}}
{{SEARCH_RESULTS}}

## Твоя задача
1. Выбери товар, который лучше всего соответствует запросу
2. Выбери 2-3 альтернативы (если есть)
3. Рассчитай нужное количество с учётом фасовки товара
4. Если ни один товар не подходит — верни selected_product_id: null

## Критерии выбора (в порядке приоритета)
1. **Соответствие запросу** — товар должен быть тем, что искал пользователь (молоко = молоко, не кефир)
2. **Наличие** — товар в наличии (InStock = true)
3. **Цена** — минимальная цена за нужный объём
4. **Фасовка** — близка к требуемому количеству (минимизируй излишек)

## Расчёт количества
Пример: пользователю нужно 2 л молока

Вариант A: Молоко 930мл за 89₽
- Нужно: ceil(2000 / 930) = 3 шт
- Получим: 2.79 л
- Стоимость: 267₽

Вариант B: Молоко 1.5л за 129₽
- Нужно: ceil(2000 / 1500) = 2 шт
- Получим: 3 л
- Стоимость: 258₽

Выбираем вариант B — дешевле за нужный объём.

## Формат ответа
Вызови инструмент select_product:
```json
{
  "name": "select_product",
  "arguments": {
    "draft_item_id": "{{DRAFT_ITEM_ID}}",
    "selected_product_id": "ID выбранного товара из списка выше",
    "quantity": 2,
    "reasoning": "Самое дешёвое молоко 1.5% в нужной фасовке",
    "alternatives": [
      {
        "product_id": "id-из-списка",
        "quantity": 3,
        "reasoning": "Знакомый бренд"
      }
    ]
  }
}
```

## КРИТИЧЕСКИ ВАЖНО
- **draft_item_id** = `{{DRAFT_ITEM_ID}}` — используй ТОЧНО это значение, НЕ МЕНЯЙ его!
- **selected_product_id** — бери ID из поля "ID:" в результатах поиска выше (например: `shampin-omy-ashan-krasnaya-ptitsa-tselye-250-g-2201800`)
- **product_id в alternatives** — также бери из результатов поиска
- Если товар не найден — верни `selected_product_id: null`
- Вызови инструмент ОДИН раз
