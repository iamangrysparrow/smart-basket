Извлеки из текста:
1. Название магазина (shop)
2. Дату покупки (date) в формате YYYY-MM-DD
3. Номер заказа/чека (order_number) если есть
4. Список товаров (items) - для каждого товара:
   - name: название товара
   - quantity: количество (число)
   - unit: единица измерения (шт, кг, л, г, мл) если указана
   - price: цена за единицу (число)
   - volume: объем/вес если указан отдельно (например "500 мл", "1 кг")
5. Общую сумму (total) если указана

ВАЖНО: Верни ответ ТОЛЬКО в формате JSON без дополнительного текста:
{
  "shop": "название магазина",
  "date": "YYYY-MM-DD",
  "order_number": "номер",
  "total": 1234.56,
  "items": [
    {
      "name": "название товара",
      "quantity": 1,
      "unit": "шт",
      "price": 100.00,
      "volume": "500 мл"
    }
  ]
}

Если какое-то поле не найдено, используй null. Если товаров не найдено, верни пустой массив items.
